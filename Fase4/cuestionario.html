Responda las siguientes preguntas


1. ¿Qué significa CSS?


a. Creative Style Sheets
b. Cascading Style Sheets
c. Computer Style Sheets

2. ¿Qué es una clase en CSS?


a. Un contenedor HTML
b. Una etiqueta HTML
c. Un identificador utilizado para aplicar estilos a un grupo de elementos HTML

3. ¿Qué es un selector de tipo en CSS?


a. Un patrón que identifica un elemento HTML en función de su tipo
b. Un patrón que identifica un elemento HTML en función de su clase
c. Un patrón que identifica un elemento HTML en función de su identificador

4. ¿Qué es Bootstrap?


a. Un Framework CSS para crear diseños responsivos y móviles
b. Una herramienta para manipular el DOM de una página web
c. Una técnica para mejorar el rendimiento de una página web
